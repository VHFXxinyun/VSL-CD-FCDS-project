Version 0.3 upgrades the communication method from GPIO level signals to USB serial communication, 
significantly enhancing data density and efficiency, while also creating strong potential for future project upgrades.
Additionally, the laser switch control has been redirected from the Raspberry Pi to the Waveshare ESP32 expansion board and is now managed via Arduino code. 
This change facilitates the encapsulation of code on the ESP32 side and reduces resource usage on the Raspberry Pi.
